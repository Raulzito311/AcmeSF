## Front-end

### Install
>/AcmeSF$ pnpm i

### Run
>/AcmeSF$ pnpm run dev

### Test
>/AcmeSF$ npx vitest
>/AcmeSF$ npx playwright test

## Back-end

### Install
>/AcmeSF/api$ composer install

### Run
>/AcmeSF/api$ php -S localhost:8080

### Test
>/AcmeSF/api$ composer test

### Analyse
>/AcmeSF/api$ composer analyse
